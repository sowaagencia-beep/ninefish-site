# Ninefish Seafood — Site institucional

Landing page institucional da **Ninefish Seafood**, desenvolvida em HTML e CSS puros
(sem frameworks ou build), com tipografia **Montserrat**.

## Estrutura

```
.
├── index.html        # Página única (header, hero, diferenciais, produtos, atacado, contato, footer)
├── styles.css        # Estilos e regras responsivas
├── FOTOS/            # Fotos de produtos e hero
├── LOGOS/            # Logo Ninefish (SVG)
└── VETORES/          # Ícones e logo AV09 (SVG)
```

## Como visualizar

Por ser estático, basta abrir o `index.html` no navegador.
Para servir localmente (opcional):

```bash
npx serve .
```

## Personalização rápida

- **Cores:** variáveis no topo do `styles.css` (bloco `:root`).
- **Catálogo:** link do botão "Baixar Catálogo" no `index.html` (hero).
- **Posição da foto do hero no mobile:** bloco comentado `POSIÇÃO DA FOTO DO HERO (MOBILE)` no `styles.css`.
